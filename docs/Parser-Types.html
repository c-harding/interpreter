data Termination Source #

type Parses a = [Maybe (String, a)] Source #

The result of a parse is internally stored as a list of optional tuples | of the remaining string and the parsed object. A value of Nothing within | the list refers to a partial parse: the string is the prefix of a valid | parse. This is, however, not returned to the user unless there are no | valid complete parses.

newtype Parser a Source #

A parser is simply a wrapper around a function transforming input to a | list of valid Parses.
